A modern, full-featured clinic management system built with ASP.NET Core MVC to streamline healthcare operations and improve patient care delivery.

## Overview

HealthCare Clinic is a comprehensive web-based management system designed to digitize and optimize clinic operations. From patient registration to appointment scheduling, this system helps healthcare providers focus on what matters most - patient care.

## Key Features

### Patient Management
- **Complete Patient Records** - Maintain detailed patient profiles with medical history
- **Quick Registration** - Streamlined patient onboarding process
- **Smart Search** - Real-time search with autocomplete functionality

### Appointment System
- **Intuitive Scheduling** - Easy-to-use appointment booking interface
- **Doctor Availability** - Dynamic time slot management based on doctor schedules
- **Specialty Filtering** - Filter doctors by medical specialty
- **Conflict Detection** - Prevent double-booking and scheduling conflicts
- **Multi-step Booking** - Guided appointment creation process
- **Status Tracking** - Monitor appointments (Scheduled, Confirmed, Completed, Canceled)

### Advanced Search & Filtering
- **Global Search** - Search across patients and appointments
- **Quick Filters** - Filter by date ranges (Today, This Week, etc.)
- **Status Filters** - Filter appointments by their current status
- **Multiple Views** - Switch between card and list presentations
- **Real-time Results** - Instant search results as you type

### Doctor Dashboard
- **My Appointments** - View and manage personal appointment schedule
- **Patient Overview** - Quick access to patient information
- **Appointment Actions** - Start consultations, view details, add notes
- **Status Management** - Update appointment statuses in real-time

### Security & Authentication
- **Role-Based Access** - Different permissions for doctors, receptionists, and admins
- **Secure Authentication** - ASP.NET Core Identity integration
- **Session Management** - Secure user session handling
